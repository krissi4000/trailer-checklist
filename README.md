# Trailer Checklist

Offline-first PWA for running family checklists at the trailer. Bilingual EN/IS. Submits each run as a row in a Google Sheet.

Live at https://krissi4000.github.io/trailer-checklist/ — deployed by GitHub Actions on every push to `master`.

```bash
npm install
npm run dev      # local server
npm test         # unit + component tests
npm run build    # production build (writes dist/)
```

Full documentation (setup, smoke test, Apps Script backend, design docs) lives in the Obsidian vault under `apps/trailer-checklist`.
